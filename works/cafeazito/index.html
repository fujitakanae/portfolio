cafeazito サイトリニューアル制作

Link:cafe azito

Overview

実際に大阪にあるカフェのサイトリニューアルです。カフェの他にもサロンやロケーションレンタルなども運営しているカフェです。

Concept

コンセプトはスロウでメロウな⼤⼈の空間です。
⼿を伸ばせばすぐ届くほど⽬の前に海を臨み、⽇本にいながら外国に来たような、⾮⽇常で特別な時間を過ごせる空間を提供するカフェです。

Tools used

Adobe XD・HTML Living Standard・CSS3・Visual Studio Code

Production time

ワイヤーフレーム・デザインカンプ　2日
コーディング　2日

In production

現存するサイトは単調なレイアウトだったので、レイアウトに動きが出るように意識して制作しました。 雰囲気を大事にしているカフェなのでお客さんに対して色々なお願い事を載せているがあまり目立たないのでWebサイトを見た人がしっかり目を通してくれるようにaboutセクションのお願い事のボタンを目立たせるようにしました。
